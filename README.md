# FairyAI — Генератор коротких сказок

Простой веб-проект, где пользователь вводит тему сказки, а сервер через API GigaChat от Сбера генерирует короткую сказку, а с помощью API Stable Diffusion XL v1.0 создает 2 иллюстрации к ней.

## Установка и запуск

1. Склонировать репозиторий:
git clone <ссылка-на-репозиторий>

2. Установка зависимостей:
npm install

3. Создать .env на основе .env.example и вставить Authorization Key:

4. Запустить сервер в директории с проектом:
node server.js

!! В целях демонстрации отключена проверка SSL-сертификата !!

P.S. из-за бесплатного API-ключа от Stable Diffusion возможны перебои с генерацией иллюстраций.
